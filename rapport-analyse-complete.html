💡 Axes d'Innovation — Ce que je propose en plus

🧠 IA Proactive vs Réactive

  • Aujourd'hui : L'IA répond aux demandes (réactive)
  • Proposé : L'IA anticipe et propose via Telegram/Push
  • Ex: "Il pleut demain → je suggère un plat réconfortant + reportez le jardinage"
  • Ex: "Vous n'avez pas planifié cette semaine → voici 3 menus rapides"
  • Principe : L'IA suggère, l'utilisateur valide. Pas de spam.

📱 Mode Companion (toujours présent)

  • FAB Chat IA permanent avec contexte de la page courante
  • Sur la page jardin : l'IA parle jardin. Sur la page budget : l'IA parle budget
  • Historique de conversation persistant par module
  • Actions suggérées dans le chat : boutons "Ajouter au planning", "Créer la liste"

📊 Insights & Analytics Personnels

  • Page "Ma famille en chiffres" : tendances, habitudes, progression sur X mois
  • Graphiques interactifs : variété des repas, assiduité routines, répartition budget
  • Comparaison temporelle : "ce mois vs dernier mois" sur chaque axe
  • Export PDF du bilan mensuel familial

🔔 Notifications Intelligentes

  • Canal unique : Telegram (sauf urgences → Push natif)
  • Fréquence adaptative : si l'utilisateur ignore, réduire. Si il réagit, maintenir
  • Groupement : 1 digest matin, 1 rappel soir (pas de spam atomique)
  • Boutons d'action directement dans Telegram (0 clic dans l'app)

🏠 Mode Automatique (Pilote IA)

  • Option "Pilote automatique" dans les préférences
  • L'IA gère : planning repas, liste courses, tâches entretien
  • L'utilisateur reçoit un récap et valide (ou active le mode 100% auto)
  • Toujours interruptible : "Cette semaine, je veux choisir moi-même"

🎯 Simplification UX — Règle des 3 clics

  • Chaque action courante doit être faisable en 3 clics max
  • "Qu'est-ce qu'on mange ce soir ?" → 1 clic dashboard → IA suggère → 1 clic valide
  • "Ajouter du lait aux courses" → ⌘K → "lait" → Entrée
  • Raccourcis rapides sur le dashboard pour actions fréquentes
