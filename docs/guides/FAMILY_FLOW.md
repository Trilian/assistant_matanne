# Guide Utilisateur - Flux Famille

Flux cible: Jules + activites + routines + budget.

## 1. Profil Jules

- Completer profil enfant (age, preferences).
- Consulter les jalons et repères du moment.
- Ajouter nouveaux jalons quand necessaire.

## 2. Activites

- Ouvrir module Famille > Activites.
- Creer une activite planifiee.
- Option suggestions IA selon contexte meteo/planning.

## 3. Routines

- Definir routines matin/soir.
- Cocher les taches effectuees.
- Ajuster frequence selon le rythme familial.

## 4. Budget Famille

- Ajouter les depenses familiales.
- Suivre statistiques mensuelles.
- Recevoir alertes en cas de depassement categories.

## 5. Achats Famille

- Creer besoins (priorite, categorie).
- Marquer comme achete une fois finalise.
- Utiliser suggestions IA si besoin.

## 6. Weekend et anniversaires

- Consulter idees weekend.
- Preparer anniversaires + checklist associee.
- Anticiper les evenements importants.
