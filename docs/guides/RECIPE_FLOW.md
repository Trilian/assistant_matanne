# Guide Utilisateur - Flux Recette

Flux cible: creation -> planification -> courses -> cuisson.

## 1. Creer une recette

- Ouvrir module Cuisine > Recettes.
- Ajouter nom, ingredients, etapes, portions.
- Option: import URL/PDF ou generation IA.

## 2. Planifier la recette

- Depuis la fiche recette: planifier sur la semaine.
- Choisir jour + repas (midi/soir).
- Verifier le planning hebdo.

## 3. Generer la liste de courses

- Depuis planning ou module Courses.
- Generer/mettre a jour la liste active.
- Ajuster quantites et priorites.

## 4. Faire les achats

- Cocher les articles pendant les courses.
- Option scan code-barres selon les produits.
- Cloturer la liste.

## 5. Mettre a jour l'inventaire

- Enregistrer les articles achetes.
- Ajuster les stocks/peremptions.
- Verifier alertes stock bas.

## 6. Cuisiner et feedback

- Suivre les etapes de la recette.
- Enregistrer un retour (a refaire / a ajuster).
- Le systeme reutilise ce retour pour les suggestions futures.
