1. Lire le `README.md` racine pour le cadrage global.
2. Consulter `docs/ARCHITECTURE.md` pour la structure technique.
3. Ouvrir ensuite le guide du domaine concerné dans ce dossier.
4. Compléter avec `docs/user-guide/README.md` pour la vue orientée usage.
