# 🎮 Guide Module Jeux

> Suivez vos paris sportifs, gérez vos grilles Loto et EuroMillions, analysez vos performances.

---

## Vue d'ensemble

Le module **Jeux** permet de suivre les activités ludiques :

| Sous-module | Accès | Description |
|-------------|-------|-------------|
| Paris sportifs | Jeux → Paris | Suivi des paris avec statistiques |
| Loto | Jeux → Loto | Gestion des grilles Loto |
| EuroMillions | Jeux → EuroMillions | Gestion des grilles EuroMillions |
| Bankroll | Jeux → Bankroll | Suivi de la bankroll et P&L |
| Performance | Jeux → Performance | Tableaux de bord et analyses |

> **Note** : Le budget jeux est **volontairement séparé** du budget famille. Les gains et pertes ne sont jamais fusionnés avec le budget familial.

---

## Paris sportifs

### Ajouter un pari

1. Allez dans **Jeux → Paris**
2. Cliquez sur **Nouveau pari**
3. Renseignez :
   - **Match** ou événement
   - **Type de pari** (simple, combiné)
   - **Cote**
   - **Mise**
   - **Bookmaker**
4. Enregistrez le pari

### Enregistrer le résultat

1. Après le match, ouvrez le pari
2. Indiquez le **résultat** (gagné, perdu, remboursé)
3. Le gain/la perte est automatiquement calculé et ajouté à la bankroll

### Statistiques de paris

La page Paris affiche :
- **Taux de réussite** global
- **ROI** (retour sur investissement)
- **Répartition** par sport/compétition
- **Évolution** de la bankroll dans le temps

## Loto

### Ajouter une grille

1. Allez dans **Jeux → Loto**
2. Cliquez sur **Nouvelle grille**
3. Choisissez vos **5 numéros** (1-49) et votre **numéro chance** (1-10)
4. Indiquez la **date du tirage**

### Vérifier les résultats

Après le tirage, le système compare automatiquement vos grilles avec les résultats officiels et affiche les numéros trouvés.

### Analyse de tendances

L'IA analyse les tirages passés pour détecter des **tendances statistiques** :
- Numéros les plus fréquents
- Numéros en retard
- Paires fréquentes

> **Note** : Les tendances sont purement statistiques et ne constituent pas des prédictions fiables.

## EuroMillions

### Ajouter une grille

1. Allez dans **Jeux → EuroMillions**
2. Cliquez sur **Nouvelle grille**
3. Choisissez vos **5 numéros** (1-50) et vos **2 étoiles** (1-12)
4. Indiquez la **date du tirage**

### Fonctionnalités similaires au Loto

- Vérification automatique des résultats
- Analyse de tendances IA
- Historique des grilles jouées

---

## Bankroll

### Suivi de la bankroll

1. Allez dans **Jeux → Bankroll**
2. Consultez :
   - Le **solde** actuel
   - L'**historique** des mouvements (mises, gains)
   - Le **P&L** (Profit & Loss) sur la période

### Évolution

Graphique d'évolution de la bankroll dans le temps, avec filtres par :
- **Période** (semaine, mois, année)
- **Type de jeu** (paris, loto, EuroMillions)

---

## Performance

### Tableaux de bord

1. Allez dans **Jeux → Performance**
2. Visualisez les KPIs :
   - **ROI** par type de jeu
   - **Taux de réussite** par sport/compétition
   - **Meilleur/pire** mois
   - **Distribution** des gains
