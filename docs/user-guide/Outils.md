# 🔧 Guide Module Outils

> Chat IA, convertisseur, météo, minuteur et notes — des outils pratiques accessibles partout.

---

## Vue d'ensemble

Le module **Outils** regroupe les utilitaires transversaux :

| Outil | Accès | Description |
|-------|-------|-------------|
| Chat IA | Outils → Chat IA | Assistant conversationnel intelligent |
| Convertisseur | Outils → Convertisseur | Conversion d'unités (masse, volume, température) |
| Météo | Outils → Météo | Prévisions météo intégrées |
| Minuteur | Outils → Minuteur | Minuteurs de cuisine multiples |
| Notes | Outils → Notes | Prise de notes rapide |

---

## Chat IA

### Utilisation

1. Allez dans **Outils → Chat IA**
2. Tapez votre question ou demande dans le champ de saisie
3. L'IA répond en tenant compte du **contexte familial** :
   - Vos recettes et préférences alimentaires
   - L'inventaire du frigo
   - Le planning de la semaine
   - L'âge de Jules (pour les adaptations)

### Mini-chat flottant

Un bouton **💬** en bas à droite de l'écran ouvre un mini-chat accessible depuis **n'importe quelle page** sans quitter le contexte actuel.

Sur mobile, le bouton redirige vers la page Chat IA complète.

<!-- Screenshot: Chat IA avec conversation contextuelle -->

### Exemples de questions

| Question | Ce que fait l'IA |
|----------|------------------|
| « Qu'est-ce qu'on mange ce soir ? » | Propose un repas basé sur le planning et l'inventaire |
| « Convertis 200g de farine en tasses » | Conversion instantanée |
| « Quel temps fait-il demain ? » | Prévisions météo locale |
| « Propose une activité pour dimanche » | Suggestion adaptée à Jules et la météo |
| « Combien j'ai dépensé ce mois ? » | Résumé budget famille + maison |

## Convertisseur

### Utiliser le convertisseur

1. Allez dans **Outils → Convertisseur**
2. Sélectionnez le **type** de conversion :
   - **Masse** : grammes ↔ onces, kg ↔ livres
   - **Volume** : ml ↔ tasses, cl ↔ fl oz, litres ↔ gallons
   - **Température** : °C ↔ °F
3. Entrez la **valeur** source
4. Le résultat s'affiche instantanément

## Météo

### Consulter la météo

1. Allez dans **Outils → Météo**
2. Consultez les prévisions :
   - **Aujourd'hui** : température, vent, précipitations
   - **Semaine** : prévisions à 7 jours
3. La météo est aussi affichée sur le **dashboard** (widget météo)

### Météo contextuelle

L'IA utilise la météo pour enrichir d'autres modules :
- **Cuisine** : recettes chaudes/froides selon la température
- **Jardin** : alertes gel, arrosage, protections
- **Activités** : suggestions adaptées indoor/outdoor
- **Entretien** : alertes tempête, canicule

## Minuteur

### Lancer un minuteur

1. Allez dans **Outils → Minuteur**
2. Cliquez sur **Nouveau minuteur**
3. Définissez la **durée** et un **nom** (ex : « Cuisson poulet »)
4. Cliquez sur **Démarrer**

### Minuteur flottant

Une fois lancé, le minuteur reste visible dans une **barre flottante** en bas de l'écran sur toutes les pages. Vous pouvez :
- **Voir** le temps restant
- **Mettre en pause**
- **Arrêter** le minuteur

Plusieurs minuteurs peuvent tourner simultanément.

## Notes

### Prendre une note

1. Allez dans **Outils → Notes**
2. Cliquez sur **Nouvelle note**
3. Écrivez votre contenu
4. La note est sauvegardée automatiquement

### Organisation

- Les notes sont triées par **date de modification**
- Utilisez la **recherche** pour retrouver une note
- Les notes sont exportables en format texte

## Recherche globale

### Palette de commandes

Depuis **n'importe quelle page**, appuyez sur `Ctrl+K` (ou `Cmd+K` sur Mac) pour ouvrir la **palette de commandes** :
- Tapez le nom d'une page pour y accéder
- Recherchez un contenu à travers tous les modules
- Consultez les 5 dernières pages visitées

### Recherche multi-entités

La recherche globale parcourt :
- Recettes
- Projets maison
- Activités
- Notes
- Contacts
- Plantes
- Documents
- Abonnements
- Tâches d'entretien

## Fonctionnalités transversales

### Export PDF

Depuis plusieurs modules, vous pouvez exporter en PDF :
- **Planning** de la semaine
- **Liste de courses**
- **Recettes** sélectionnées
- **Budget** mensuel
- **Journal familial**
- **Rapport mensuel**

### Export iCal

Les événements du planning sont exportables au format iCal pour synchronisation avec Google Calendar, Apple Calendar, etc.

### Export CSV

Les tableaux de données peuvent être exportés en CSV via le bouton **Export CSV** présent sur les pages de listes.
